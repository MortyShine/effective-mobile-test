# effective-mobile-test

Удаление товара: Согласно п.2, товар удаляется кнопкой «удалить», а минимальное кол-во — 1 шт. В п.9 указано, что товар удаляется автоматически при уменьшении счетчика до 0.
Исправление — объединить пункты: товар удаляется как специальной кнопкой, так и при нажатии на «минус», если текущее количество равно 1.

Лимиты: П.4 (общий лимит 20 шт.) противоречит суммарному объему из п.1 и п.3 (50 шт.), что путает пользователя.Исправление — исключить п.4. Оставить только лимиты на количество позиций (5) и количество единиц внутри одной позиции (10 шт.).

Текст ошибки: В п.6 указано общее уведомление «Лимит корзины превышен», но пользователю будет неясно, какое именно правило нарушено.
Исправление — детализировать тексты ошибок для каждого лимита (по позициям, по количеству одного SKU, по общему количеству в корзине).

Нумерация: В перечне требований пропущен п.12.
